# End User Guide: How to Use Qdrant CMS

This guide is for end users (not developers). It provides step-by-step instructions and screenshots to help you use Qdrant CMS.

## Getting Started

1. Access the CMS: Navigate to the Qdrant CMS application URL provided by your administrator.
2. Login/Register: If required, log in or register using your credentials.
3. Dashboard Overview: After login, you will see the main dashboard with a sidebar, search bar, and content area.

![Qdrant CMS dashboard screenshot](./images/dashboard.png)

## Common Actions

- Browse Content: Use the sidebar to view available collections or items.
- Search: Use the search bar to find specific entries.
- View Details: Click on any item to view its details.

## Creating New Entries

1. Click the "Add New" button or similar action in your collection.
2. Fill in the required fields using the form provided. Typical fields include title, description, tags, and any media uploads.
3. Save your entry by clicking the "Save" or "Create" button. You should see a confirmation message when the entry is saved.

![Entry creation form screenshot](./images/create-entry.png)

## Editing or Deleting

- To edit: Select an entry and click "Edit." Update fields as needed and save changes.
- To delete: Select an entry and click "Delete." Confirm your action when prompted. Deleted entries may not be recoverable depending on configuration.

## Media and Images

- Upload images or files when creating or editing an entry using the media upload field.
- Follow any filesize or format restrictions shown in the upload dialog.

## Best Practices

- Use clear, concise titles and descriptive content for better searchability.
- Add tags or categories to help organize content.
- Preview changes before publishing if a preview feature is available.

## Getting Help

- Check the Help or Documentation link in the sidebar for more detailed instructions.
- If you encounter issues, open a support issue in the project repository and include screenshots and steps to reproduce the problem.
